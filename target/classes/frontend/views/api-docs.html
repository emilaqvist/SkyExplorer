SkyExplorer API Documentation

API documentation for the SkyExplorer flight and weather search service

SkyExplorer API Overview

A RESTful API for searching flights and retrieving weather information

The SkyExplorer API provides a unified interface for searching flights between locations and retrieving weather forecasts for destinations. It aggregates data from multiple sources including SkyScanner for flight information and Tomorrow.io for weather data.

Base URL

http://localhost:7000

Authentication

The API currently does not require authentication for requests.

Rate Limiting

There are no specific rate limits imposed by the SkyExplorer API itself, but be aware that the underlying external APIs (SkyScanner and Tomorrow.io) have their own rate limits.

Response Format

All API responses are returned in JSON format. Successful responses will contain the requested data, while error responses will include an error message and appropriate HTTP status code.

API Endpoints

Available endpoints and their parameters

GET /search

Search for flights between locations and retrieve weather information for the destination.

Query Parameters

Parameter Type Required Description
from string Yes Departure city or airport code
destination string Yes Destination city or airport code
departDate string Yes Departure date in YYYY-MM-DD format

Response

Returns a JSON object containing flight search results and weather forecast for the destination.

Response Fields

Field Type Description
flights array List of flight results matching the search criteria
weather object Weather forecast data for the destination

Status Codes

Code Description
200 Success - Returns flight and weather data
400 Bad Request - Missing or invalid parameters
404 Not Found - No flights found for the specified route
500 Internal Server Error - Error occurred while processing the request

Represents a single flight search result with details about the flight.

Field Type Description
id string Unique identifier for the flight
price number The price of the flight
formattedPrice string Formatted price with currency symbol
departureCity string City of departure
arrivalCity string City of arrival
departureTime string Departure time in ISO format
arrivalTime string Arrival time in ISO format
duration number Flight duration in minutes
stops number Number of stops
airline string Name of the airline
departureAirport string Name of the departure airport
arrivalAirport string Name of the arrival airport

Represents weather forecast data for a location.

Field Type Description
timelines object Contains weather timelines
timelines.hourly array Hourly weather forecast data
timelines.hourly[].time string Timestamp for the forecast in ISO format
timelines.hourly[].values object Weather values for this time period
timelines.hourly[].values.temperature number Temperature in Celsius
timelines.hourly[].values.weatherCode number Code representing weather conditions

Weather Codes

Code Description
1000 Clear, Sunny
1100 Mostly Clear
1101 Partly Cloudy
1102 Mostly Cloudy
1001 Cloudy
4000, 4200 Light Rain, Drizzle
4001, 4201 Rain, Heavy Rain
5000, 5100 Snow, Light Snow
5101 Heavy Snow
8000 Thunderstorm

Using the API with JavaScript

// Example of how to call the API using JavaScript fetch
async function searchFlightsAndWeather() {
  const departureCity = "Stockholm";
  const destinationCity = "London";
  const departureDate = "2023-12-25";

  try {
    const url = `http://localhost:7000/search?from=${encodeURIComponent(departureCity)}&destination=${encodeURIComponent(destinationCity)}&departDate=${encodeURIComponent(departureDate)}`;

    const response = await fetch(url);
    if (!response.ok) {
      throw new Error("Network error: " + response.status);
    }

    const data = await response.json();
    console.log("Flights:", data.flights);
    console.log("Weather:", data.weather);

    // Process the data...
  } catch (error) {
    console.error("Error calling API:", error);
  }
}
